Goals for the Project

This is a work in progress with the aim to create a wepage centered around a map pinpointing areas of Central Eurasia complete with links to the phonological inventories of the regions pinpointed on google maps.

Currently, the webpage has a hime page intended to house the map. It is complete with a dropdown menu that has options linking the homepage to other notable resorces.

The next step is to embed Google maps into the page using JavaScript API in order to access custom markers and otther tools that enable seamless user interaction and tap into more customazation. Then, improvement of the front end layout for the auxilliary pages will be completed. Following this, the pertinent content will be added to those pages. Another idea is to make this page translatable to Kazakh, Turkish, or both.
