Goals for the Project

This is a work in progress aimed at creating a webpage centered around an interactive map of Central Eurasia. The map will pinpoint specific regions and provide links to the phonological inventories associated with the languages spoken there.

Currently, the website includes a homepage intended to host the map. It also features a dropdown navigation menu that links to additional resources and auxiliary pages.

The next step is to embed Google Maps into the webpage using the JavaScript API in order to access custom markers and other tools that support seamless user interaction and greater customization. After the map integration is complete, the front-end layout of the auxiliary pages will be refined and expanded. Relevant linguistic content and resources will then be added to those pages. Another future goal is to make the website translatable into Kazakh, Turkish, or both in order to improve accessibility for a wider audience.

Summary

  • Use JavaScript API
  • Add custom markers
  • Improve user interaction
  • Refine page layout
  • Add linguistic content
  • Add auxiliary pages
  • Enable translations (Kazakh/Turkish)
